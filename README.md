Abre Git Bash en la carpeta de tu proyecto (puedes hacer clic derecho en la carpeta y seleccionar "Git Bash Here").
Verifica cambios: Usa git status para ver los archivos modificados.
Añade los archivos: Usa git add . para preparar todos los cambios.
Confirma los cambios: Ejecuta git commit -m "Descripción de los cambios".
Sube a GitHub: Ejecuta git push origin main (reemplaza main por tu rama, ej. master).
